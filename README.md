# Andragogy Learning Bot

WhatsApp-based Mobile Learning Platform

## System Components
- **Andragogy** — WhatsApp chatbot for learners
- **Andragogy Insight** — Flutter facilitator dashboard

## Tech Stack
- Backend: Node.js + Express
- Database: MongoDB Atlas
- Dashboard: Flutter (Dart)
- Messaging: Twilio WhatsApp API
- Tunnel: Ngrok

## How to Run
1. cd backend → npm run dev
2. ngrok http 5000
3. cd flutter → flutter run -d chrome
